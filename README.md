## Development notes
- Edit HTML/CSS in the `css/`, `resume/`, and `projects/` folders.
- Styles are organized in `css/style.css` (backup `css/style.css.bak` exists).
- Contact form uses a Formspree placeholder in the `sendEmail()` helper — replace with your Formspree ID before enabling.
